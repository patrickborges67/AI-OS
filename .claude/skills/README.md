# Skills Locais

Crie skills locais deste workspace nesta pasta, uma por diretorio:

```text
.claude/skills/nome-da-skill/
  SKILL.md
  references/
  scripts/
```

## Skills incluidas

- `carrossel` - carrosseis editoriais em HTML/PNG.
- `carrossel-essentialweb` - skill especifica da EssentialWeb, com assets e referencias da marca.
- `ads-ratos` - inteligencia de trafego pago.
- `meta-ads-ratos` - execucao Meta Ads.
- `google-ads-ratos` - execucao Google Ads.
- `ga4-ratos` - consulta GA4.
- `gpt-image-2` - geracao de imagens.
- `publicar-social-ratos` - publicacao social.
